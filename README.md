# Diyabet-Veri-Seti-ve-Lojistik-Regresyon-ile-Diyabet-Hastaligi-Teshisi
Bu proje, Lojistik Regresyon kullanarak diyabet hastalığını tahmin etmeyi amaçlar. Model, [Örn: Pima] Diyabet Veri Seti üzerinde Scikit-learn ile eğitilmiş ve hastaların sağlık verilerine (glikoz, BMI, yaş vb.) dayalı bir sınıflandırma yapısı sunar.
